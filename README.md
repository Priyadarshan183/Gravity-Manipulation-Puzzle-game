# Gravity-Manipulation-Puzzle-game
It is a gravity manipulation game

GAME CONTROLS

Movement: - W / A / S / D → Move character 
- Mouse → Look around (camera control) 
- Shift → Run 
- Space → Jump

Gravity Manipulation: - Arrow Keys → Preview gravity direction 
- Up Arrow → Gravity Back 
- Down Arrow → Gravity Forward 
- Left Arrow → Gravity Left 
- Right Arrow → Gravity Right 
- Q → Flip gravity upward (ceiling) 
- Enter → Confirm gravity change 
- Escape → Cancel gravity preview

OBJECTIVE

-   Collect all cubes in the level before time runs out.

WIN CONDITION

-   Collect all cubes in the level.
-   When all cubes are collected → YOU WIN.

LOSE CONDITIONS

1.  Time Up:
    -   If the timer reaches zero → YOU LOSE.
2.  Fell Too Long:
    -   If the player drifts away from the game play area with no surface to land → YOU LOSE.

GAMEPLAY NOTES

-   Gravity can be changed to walk on walls and ceilings.
-   Use gravity smartly to reach difficult areas.
-   Avoid staying airborne for too long.
-   Keep track of remaining time and cubes.
